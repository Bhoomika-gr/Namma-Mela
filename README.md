# Namma-Mela

## Overview
Namma-Mela is an Android application developed to support rural theater and village drama events digitally. The app helps organizers promote local events and allows users to view event details, posters, cast information, and seat availability.

## Problem Statement
Many rural cultural events and village dramas are promoted only offline, making it difficult for people to know event details. Namma-Mela provides a simple digital platform for promoting and managing these events.

## Features
- View village drama events
- See cast and event details
- Check seat availability
- Easy navigation

## Tech Stack
- Kotlin
- Android Studio

## Installation Steps
1. Clone the repository
2. Open project in Android Studio
3. Sync Gradle files
4. Run the application
